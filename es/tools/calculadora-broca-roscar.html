Calculadora de broca para roscar

¿Qué es el diámetro de broca para roscar?

Es el tamaño de taladro previo al machueleado: algo menor que el diámetro mayor del tornillo para que el macho forme la rosca con profundidad adecuada.

Elija una rosca métrica y, si lo desea, ajuste el paso. El diámetro de broca recomendado se actualiza al instante.
