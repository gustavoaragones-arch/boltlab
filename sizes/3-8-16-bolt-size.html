3/8-16 Bolt Size and Thread Specifications

3/8-16 is a common imperial (UNC) bolt size for automotive, machinery, and structural use. A 3/8 inch nominal diameter with 16 threads per inch provides strong threads and is standard in many North American applications. Spanish reference: perno 3/8-16. Metric vs UNC thread size

Metric threads list diameter in millimeters and pitch in millimeters per thread; UNC lists nominal inch diameter and threads per inch. They are not interchangeable—taps, dies, and gauges differ.

How to choose the right bolt size

Choosing the correct bolt size depends on load, material, and thread type. Larger diameters increase strength, while thread pitch affects holding force.

FAQ

What does 3/8-16 mean?

3/8 is the nominal diameter in inches (0.375 in). 16 is the number of threads per inch (TPI). It is a common UNC coarse thread.

What drill size for 3/8-16 tap?

Use a 5/16 in drill for a standard 3/8-16 tap.

What metric bolt is closest to 3/8-16?

M10 is the closest metric equivalent to 3/8-16. See M10 bolt size for full specs.
