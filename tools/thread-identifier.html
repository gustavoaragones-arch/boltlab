Thread Identifier Tool

Quick answer

A thread identifier ranks likely standards from measured major diameter and pitch or TPI inputs.

It is a desk check before gauges, not a replacement for metrology on safety-critical joints.

Use the form below, then open tap drill size tools and screw size conversion when you pick hardware.

Identify metric or imperial threads by entering diameter and pitch, then compare candidates with thread types diagrams.

Match tap drill size outputs to the sizes you select here.

Identify thread standard

How matching works

This identifier checks your entered values against common thread standards—useful after screw identifier visual narrowing—and returns matches within diameter ±5% and pitch ±5%.
