Thread Pitch to TPI Converter

Quick answer

Thread pitch in millimeters is the axial distance per thread; TPI counts threads per inch.

Drawings may show either unit, so conversions prevent mismatched taps and gages.

Use the converter below, then read thread types when you separate UNC from metric coarse.

Switch mode and type your value. Result updates immediately with no submit button.

Use thread types for naming when drawings mix series labels.

Use the thread identifier when matching unknown parts.

Converter

Formula used

TPI = 25.4 / pitch(mm) and pitch(mm) = 25.4 / TPI.
