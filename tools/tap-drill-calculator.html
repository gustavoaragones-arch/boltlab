Tap Drill Calculator

What is tap drill size?

Tap drill size is the pilot hole diameter you drill before cutting threads with a tap. It is smaller than the bolt’s major diameter so the tap forms usable thread depth.

Pick a metric thread size and optionally adjust pitch. Recommended tap drill size updates instantly.

Calculation method

For metric taps, a common workshop formula is major diameter minus pitch. This tool also shows coarse reference values from a lookup table.
